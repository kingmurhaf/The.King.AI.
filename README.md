## 🚀 خطوات الرفع على GitHub Pages

### 1. أنشئ حساباً على GitHub
- اذهب إلى [github.com](https://github.com) وأنشئ حساباً مجانياً

### 2. أنشئ Repository جديداً
```
Repository name: king-ai
✅ Public
✅ Add a README file (اختياري)
```

### 3. ارفع الملفات
**الطريقة السهلة (بدون Git):**
1. افتح الـ Repository
2. اضغط **Add file → Upload files**
3. اسحب جميع الملفات دفعة واحدة
4. اضغط **Commit changes**

**أو عبر Git:**
```bash
git clone https://github.com/اسمك/king-ai.git
cd king-ai
# ضع الملفات هنا
git add .
git commit -m "🚀 King AI v25.0 — Full PWA"
git push origin main
```

### 4. فعّل GitHub Pages
1. اذهب إلى **Settings** في الـ Repository
2. من القائمة الجانبية اختر **Pages**
3. تحت **Source** اختر: **GitHub Actions**
4. احفظ

### 5. انتظر الـ Deploy
- اذهب إلى تبويب **Actions**
- انتظر حتى يكتمل (دقيقة تقريباً)
- سيظهر رابط مثل: `https://اسمك.github.io/king-ai/`

## 📱 تثبيت التطبيق على الهاتف

### أندرويد (Chrome):
1. افتح الرابط في Chrome
2. ستظهر رسالة "أضف إلى الشاشة الرئيسية" في الأسفل
3. أو اضغط ⋮ ← "تثبيت التطبيق"
4. اقبل الأذونات المطلوبة

### iOS (Safari):
1. افتح الرابط في Safari
2. اضغط زر المشاركة ↑
3. اختر "إضافة إلى الشاشة الرئيسية"
4. اضغط "إضافة"

## ⚙️ الأذونات المطلوبة

عند أول فتح، سيطلب التطبيق:

1. **الإشعارات** — لإرسال المنبهات حتى عند إغلاق التطبيق
2. **المزامنة في الخلفية** — لفحص المنبهات كل دقيقة
3. *(اختياري)* **الموقع** — لأوقات الصلاة والطقس

## 🔔 كيف تعمل المنبهات في الخلفية؟

```
التطبيق يحفظ التذكيرات ← SW يستقبلها ← يفحص كل 30 ثانية
← يُرسل إشعار نظام ← يظهر حتى لو الهاتف مقفل
```

**الـ Service Worker** يعمل في الخلفية حتى بعد:
- إغلاق التطبيق ✅
- إطفاء الشاشة ✅
- (ليس بعد إعادة تشغيل الهاتف — هذا قيد النظام)
